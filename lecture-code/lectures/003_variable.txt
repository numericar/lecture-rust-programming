Variable
    เป็นสิ่งที่เอาไว้เก็บข้อมูล โดยค่าเริ่มต้นจะไม่สามารถเปลี่ยนแปลงข้อมูลได้ (immutable)
    ถ้าต้องการให้เปลี่ยนเแปรงข้อมูลได้จะต้องระบุ mut (mutable) ลงไปด้วย

    let a = 5;
    เปลี่ยนแปลงไม่ได้

    let mut b = 10;
    b = 20;
    เปลี่ยนแปลงได้

    โดยเหตุผลที่จะต้องเป็น Immutable เพื่อป้องกันการเปลี่ยนแปรงข้อมูลโดยที่ไม่ได้ตั้งใจ
    และป้องกันการเข้าถึงข้อมูลพร้อมกัน (data race)
