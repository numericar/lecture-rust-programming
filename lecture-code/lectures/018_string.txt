String
    เป็นหนึ่งในชนิดข้อมูลสำหรับจัดเก็บ ข้อความ ซึ่งสามารถเปลี่ยนแปลงขนาดได้ (dynamic size)
    ต่างจาก &str ที่เป็น string slice ซึ่งมีขนาดคงที่และใช้การอ้างอิงไปยังข้อมูลที่อยู่ในหน่วยความจำ

    ** String: ข้อมูลที่จัดการหน่วยความจำเอง สามารถเพิ่มหรือลดขนาดได้
    ** &str: ข้อมูลที่อ้างอิงข้อความ เช่น ตัวอักษรใน literal string

    ความสามารถของ String 
        1. dynamic size 
        2. Ownership: มี Ownership ของข้อมูล และจัดการหน่วยความจำเอง 
        3. UTF-8 Encoding: เป็นการเข้ารหัสแบบ utf-8 ทำให้รองรับได้หลายภาษา
