Data Type
    เป็นการระบุชนิดของข้อมูลให้กับตัวแปร
    โดยจะแบ่งเป็น 2 กลุ่ม
        1. Scalar Types     เป็นประเภทข้อมูลที่จะเก็บ 1 ค่า เช่น Integer, Floating-Point, Boolean, Character
                                1.1 Interger    ใช้สำหรับเก็บเลขจำนวนเต็ม โดยรองรับหลายแบบ เช่น i32, u8, i64
                                1.2 Floating    ใช้สำหรับเก็บเลขจำนวนเต็มที่มีทศนิยม เช่น f32, f64
                                1.3 Boolean     ใช้เก็บค่าจริงหรือเท็จ
                                1.4 Character   ใช้เก็บตัวอักษร 1 ตัว
        2. Compound Types   เป็นประเภทข้อมูลที่ใช้เก็บได้หลาย ๆ ค่าเข้าด้วยกัน โดยมี 2 ประเภทค่อ Tuples, Arrays
                                2.1 Tuples      เก็บข้อมูลได้หลายประเภทและหลายตัวพร้อมกัน
                                2.2 Arrays      เก็บข้อมูลได้หลายตัว แต่ได้แค่ประเภทเดียว

    และยังมีชนิดข้อมูลที่มาจาก Standard Library ซึ่งมักจะใช้บ่อย ๆ เช่น 
        1. String   เป็นชนิดข้อมูลที่เก็บตัวอักษรได้หลายตัว
        2. &str     อ่านว่า string slice เป็นการอ้างอิงไปที่ String แบบความยาวคงที่ ไม่สามารถเปลี่ยนแปรงได้
        3. Vector   ใช้สำหรับจัดเก็บข้อมูลหลายค่า และสามารถเพิ่มหรือลดจำนวนได้ตามต้องการ
                    ใช้สัญลัก Vec<T>
