Generic
    เป็นกลไกที่ช่วยให้เราสามารถเขียน code ที่รอบรับหลายชนิดข้อมูลได้ โดยไม่ต้องเขียนโค้ดซ้ำซ้อน

    โดยมีหลักการดังนี้
        1. Parametric Polymorthism: ช่วยให้ชุดคำสั่งสามารถทำงานได้ โดยไม่ต้องระบุชนิดข้อมูลล่วงหน้า
        2. rust จะ monomorphize code (แปลงเป็นชนิดข้อมูลเป้าหมาย) ในขั้นตอนการ compile ทำให้
            ประสิทธิภาพไม่ลดลง

    โดยเป้าหมายของการทำ Generic คือ 
        1. Code Reusablility: ช่วยให้เราเขียนโค้ดที่นำกลับมาใช้ซ้ำได้ โดยไม่ต้องระบุชนิดข้อมูลหลายครั้ง
        2. Type Safety: Rust จะตรวจสอบชนิดของข้อมูลในจังหวะ compile ทำให้สามารถหลีกเลี่ยงข้อผิดพลาดได้
        3. Performance: Generic ใน rust ถูก monomorphize ในขั้นตอนการ compile file ซึ่งทำให้ไม่มี
                        Overhead ใน runtime เกิดขึ้น
