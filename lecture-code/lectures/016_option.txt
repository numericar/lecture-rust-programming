Option<T>
    ใช้สำหรับจัดการกับข้อมูลที่อาจจะมีหรือไม่มี มีความคล้ายกับ Result<T, E> แต่ถูกออกแบบมาให้ใช้ในสถาณการที่ไม่
    จำเป็นจะต้องจัดการกับข้อผิดพลาด

    รูปแบบของ Option<T>
        enum Option<T> {
            Some(T),
            None
        }

    Some(T): หมายถึง มีข้อมูลอยู่ โดยข้อมูลจะถูกเก็บไว้ที่ T 
    None: หมายถึง ไม่มีข้อมูล (null)

    หากเรามั่นใจว่าค่าจะไม่มีทางเป็น None สามารถใช้ .unwrap ได้
