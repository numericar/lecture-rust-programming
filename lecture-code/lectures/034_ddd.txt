Entity vs Value Object
    Entity (database)
        - สามารถเปลี่ยนแปรงได้ตามการเวลา
        - มีความเป็นเอกลักษณ์ในตัวเอง
        - สามารถติดตามไปได้เรื่อย ๆ แม้จะเปลี่ยนแปลงไป

    Value Object (fix code, config file)
