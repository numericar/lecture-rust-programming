Associated Types 
    เป็นคุณสมบัติของ trait ที่ช่วยให้เราสามารถกำรหนดประเภทที่เกี่ยวของกับ trait ได้ ช่วยลดความซับซ้อน
    ในการใช้งาน generic

    ทำไมต้องใช้ Associated Types ?
        1. ช่วยลดความยุ่งยากในการใช้ระบุ generic types หลายตัว
        2. ทำให้ชุดคำสั่งอ่านง่าย โดยกำหนดประเภทที่เกี่ยวข้องภายใน trait

    โดยวิธีใช้งานคือใช้คำสั่ง type ภายใน trait

    ** การใช้งาน Associated จะไม่เพิ่มจำนวนของ code ในขั้นตอนการ compile เพราะประเภทถูกระบุไว้แล้ว
