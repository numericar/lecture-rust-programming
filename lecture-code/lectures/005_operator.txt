Operator
    เป็นสัญลักษณ์ที่ใช้ดำเนินการกับค่าต่าง ๆ

    Arithmetic Operators
    ตัวดำเนินการทางคณิตศาสตร์
        1. +    นำค่ามาบวกกัน 
        2. -    นำค่ามาลบกัน
        3. *    นำค่ามาคูณกัน
        4. /    นำค่ามาหารกัน
        5. %    นำค่ามาหารกันและเอาเศษมาเห็นผลลัพธ์

    Comparison
    ตัวดำเนินการเปรียบเทียบ
        1. ==   ค่าทั้งคู่เท่ากัน
        2. !=   ค่าทั้งคู่ไม่เท่ากัน
        3. >    ค่าทางซ้าย มากกว่า ทางขวา
        4. <    ค่าทางซ้าย น้อยกว่า ทางขวา
        5. >=   ค่าทางซ้าย มากว่าหรือเท่ากับ ค่าทางขวา
        5. <=   ค่าทางซ้าย น้อยกว่าหรือเท่ากับ ค่าทางขวา

    Logical Operator
    ตัวดำเนินการทางตรรกะศาสตร์
        1. &&   (AND) ซ้าย และ ขวา เป็นจริง
        2. ||   (OR) ซ้าย หรือ ขวา เป็นจริง
        3. |    (NOT) กลับจากจริงเป็นเท็จ เท็จเป็นจริง

    Assignment Operators 
    ตัวดำเนินการดำหนดค่า
        1. =    กำหนดค่าให้กับตัวแปร
        2. +=   นำค่าของตัวแปรที่จะกำหนด มาบวกกับค่าที่ระบุ และเก็บลงในตัวแปรที่กำหนด
        3. -=   นำค่าของตัวแปรที่จะกำหนด มาลบกับค่าที่ระบุ และเก็บลงในตัวแปรที่กำหนด
        4. *=   นำค่าของตัวแปรที่จะกำหนด มาคูณกับค่าที่ระบุ และเก็บลงในตัวแปรที่กำหนด
        5. /=   นำค่าของตัวแปรที่จะกำหนด มาหารกับค่าที่ระบุ และเก็บลงในตัวแปรที่กำหนด
        6. %=   นำค่าของตัวแปรที่จะกำหนด มาหารเอาเศษกับค่าที่ระบุ และเก็บลงในตัวแปรที่กำหนด
