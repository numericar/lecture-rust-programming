Refernce
    คือการยืมค่าผ่านการอ้างอิง โดยเราสามารถใช้เครื่องหมาย '&' เพื่อสร้าง refernce สำหรับค่าที่เราต้องการใช้ชั่วคราว
    โดยไม่ต้องย้ายความเป็นเจ้าของ

    * Immutable Reference (&T)      ยืมแบบอ่านอย่างเดียว ไม่สามารถแก้ไขได้
    * Mutable Reference (&mut t)    ยืมแบบที่สามารถแก้ไขข้อมูลได้ด้วย โดยสามารถยืมได้เพียงครั้งเดียวในช่วงเวลาเดียวกัน

    

สรุป 
    Ownership: แต่ละค่าจะมีเจ้าของชัดเจน เมื่อ scope ของเจ้าของสิ้นสุด ค่านั้นจะถูกปล่อยออกจากหน่วยความจำ
    Borrowing: การยืมค่าทำได้ผ่านการอ้างอิง โดยเราสามารถยืมได้ทั้งแบบ Mutable และ Immutable
    Reference: เป็นการยืมชั่วคราวโดยไม่ต้องย้ายเจ้าของ
